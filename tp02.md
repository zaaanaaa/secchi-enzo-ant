# Face Mask Detection


**Dataset:** https://www.kaggle.com/datasets/andrewmvd/face-mask-detection

---

**¿Qué predecirían?**
Predeciría si una persona en una imagen tiene el barbijo bien puesto, mal puesto (por ejemplo sin cubrir la nariz) o directamente no tiene. Esto ya no se usa tanto, pero en un tiempo como la pandemia, podria haber sido usado para controlar.

**¿Qué tipo de ML usarían?**
Usaría aprendizaje supervisado. El dataset ya viene con las imágenes etiquetadas y con bounding boxes marcando cada cara, así que está pensado para ese tipo de enfoque.

**¿Qué features les parecen más útiles y por qué?**
Los features más importantes son las regiones de las caras dentro de cada imagen, especialmente la zona de nariz y boca, que es lo que determina si el barbijo está bien o mal usado. También influyen cosas como la iluminación y la calidad de la imagen, porque si las condiciones son malas el modelo puede confundirse y clasificar mal.
